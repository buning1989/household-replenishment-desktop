# 403家庭管家

面向家庭日常事务的 Windows / macOS 桌面助手。当前从补货提醒切入，界面使用 React 开发，Electron 提供托盘、空闲检测和系统通知能力。

## 当前已包含

- 按“补货周期 - 提前量”判断待补货状态
- 充足 / 快用完 / 急需补货三档状态
- 首页待办优先，家中清单按方形分组卡片展示
- 学习型 / 固定型物品
- 补货历史与平滑校准建议，建议需用户确认后才采用
- 添加、编辑、自定义品类、补货链接和价格记录
- 已补到家、稍后提醒、勿扰时段、30/60 分钟重复提醒
- macOS 菜单栏常驻、系统空闲检测、合并桌面通知
- 浏览器预览和 Electron 桌面预览

双电脑同步服务尚未接入，当前数据保存在本机。数据模型已经保留 `updatedAt` 和稳定 ID，下一阶段可直接增加同步层。

## 开发预览

```bash
npm install
npm run dev
```

命令会同时启动 Vite 和 Electron，随后弹出桌面应用窗口。只预览界面可运行：

```bash
npm run dev:web
```

然后访问 `http://127.0.0.1:5173`。浏览器模式不包含托盘、系统通知和空闲检测。

## 构建

```bash
npm run build
```

本机生成未签名的 macOS 安装包（适合自己的电脑测试）：

```bash
npm run package:mac
```

正式发给其他用户前，还需要 Apple Developer 证书完成签名与公证。
